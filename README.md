# Template Node ETML

Template de base pour les projets Node.js à l'ETML.

## Prérequis

- VS Code
- Docker Desktop

## Utilisation

1. Cloner le dépôt
2. Ouvrir le dossier dans VS Code
3. Cliquer sur "Reopen in Container"

## Lancer le projet

npm start
