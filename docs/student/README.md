# Guide étudiant – Projet Node ETML

## Objectif

Ce projet sert de base pour les exercices et projets Node.js à l'ETML.

## Structure du projet

src/ → code source  
tests/ → tests automatisés  
docs/ → documentation  

## Démarrage

Installer les dépendances :

npm install

Lancer le projet :

npm start

## Vérifier le code

npm run lint

## Règles importantes

- Respecter la structure du projet
- Faire des commits atomiques
- Respecter les conventions ETML
