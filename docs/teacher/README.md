# Guide enseignant – Template Node ETML

Ce template permet de standardiser les projets Node.js à l'ETML.

## Objectifs pédagogiques

- environnement identique pour tous les élèves
- apprentissage des bonnes pratiques Git
- introduction aux outils professionnels (ESLint, Prettier)

## Contenu du template

- Dev Container Node
- ESLint
- Prettier
- structure projet standardisée
- conventions de commits ETML

## Utilisation

Le template peut être utilisé :

- directement sur GitHub
- via GitHub Classroom
